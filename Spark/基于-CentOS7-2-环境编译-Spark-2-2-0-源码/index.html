前期准备

  1. Java8 的安装

  2. Maven3.3.9 的安装

下载maven后进行环境变量的设置，设置maven的内存使用,在环境变量中加入如下命令

export MAVEN_OPTS="-Xmx2g -XX:ReservedCodeCacheSize=512m"

  2. git：直接输入命令：sudo yum install git 下载 git
